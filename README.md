# Student Registry Smart Contract

This is for the OpenCode Induction Task (Beginner)

This project is a basic Solidity smart contract that allows a user to register as a student and retrieve student details using their wallet address.

The contract is deployed and tested using **Remix IDE** with **MetaMask** on the **Sepolia test network**.
